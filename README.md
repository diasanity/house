# 海悅廣告｜土地評估報告產生器

這是一個用於代銷土地評估的一頁式報告產生器。核心目標是讓使用者可以把 ChatGPT 依固定土地調研邏輯產出的 JSON 直接貼入 App，並在畫面中逐欄位手動修改，最後列印或另存成 PDF。

## 主要功能

- 貼上 ChatGPT 產出的土地評估 JSON
- 一鍵匯入 JSON 並同步到表單欄位
- 所有報告欄位皆可手動修改
- 右側即時預覽「海悅廣告 土地評估分析表」版型
- 複製目前報告 JSON
- 下載目前報告 JSON
- 使用瀏覽器列印功能輸出 PDF

## 報告欄位

- 配合業主
- 調研時間
- 標的位置
- 標的地號
- 土地分區
- 基地面積
- 法定建蔽率
- 法定容積率
- 臨路條件
- 土地售價
- 學區
- 里別
- 基地四向現況：北向、西向、南向、東向
- 交通動線
- 生活機能
- 公共建設
- 區域銷況
- 建議產品
- 個案參考
- 價格預判
- 優勢三點
- 劣勢三點
- 初步結論
- 資料來源／待複核事項

## 使用方式

```bash
npm install
npm run dev
```

開啟 Vite 顯示的網址後即可使用。

## ChatGPT JSON 使用流程

1. 在 ChatGPT 輸入土地評估指令。
2. 要求 ChatGPT 依固定格式輸出 JSON。
3. 複製完整 JSON。
4. 貼到 App 左側「ChatGPT JSON 貼上區」。
5. 點選「匯入 JSON」。
6. 在中央表單逐欄位修改。
7. 確認右側預覽無誤。
8. 點選「列印/PDF」輸出報告。

## 土地調研邏輯摘要

- 基地面積需逐筆查詢地號面積並換算坪數，坪數公式為：平方公尺 × 0.3025 = 坪。
- 土地分區、建蔽率、容積率需優先查都市計畫書、細部計畫書、土地使用分區管制要點與政府公開資料。
- 基地四向現況需依實際地圖、衛星圖、街景、地籍圖或使用者提供圖資判讀，臨路需寫路名與路寬，臨房需寫型態與樓層。
- 學區需查教育局最新年度學區公告，國小與國中分開確認。
- 里別需透過地籍位置、里界圖、戶政或區公所資料交叉判斷。
- 競案調查以基地附近、客戶實際會比較的個案為優先；第一順位為線上預售案，預售案不足時依序補屋齡 5 年內、5 至 10 年、10 至 15 年個案。
- 成交行情原則以內政部實價登錄成交均價為核心，先抓近半年，若近半年無成交則改抓近一年。
- 車位價格以周邊競案實際車位行情為核心，公式推算僅作合理性檢查。
