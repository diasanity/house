土地評估手機版應用程式

實務流程：只輸入建設公司＋地段地號；由 ChatGPT 完成調研後匯入結果，每一欄皆可修改。

你實際只需要輸入這兩項

之後你給我「建設公司名稱＋土地地段地號」，我會負責調研其他欄位。調研完成後，我會輸出一份可貼入此工具的 JSON，工具會自動帶入表格。

調研時間會自動抓今天日期。完成後請按「產生調研指令」，再複製貼給 ChatGPT。

貼上 ChatGPT 調研結果

請將 ChatGPT 回傳的 JSON 貼在下方，按「套用調研結果」。套用後每個欄位都能再修改。

競案資料匯入與自動篩選

可匯入內政部實價登錄整理後的 Excel / CSV。系統會依「預售案 → 屋齡5年內 → 5到10年 → 10到15年」補足4個有效競案；近一年無成交者會自動剔除。

建議欄位：案名、建設公司、類型、屋齡、距離、交易日期、成交單價、房型、坪數、車位價格、資訊來源、備註。成交單價請使用實價登錄成交資料，不採預售開價。
